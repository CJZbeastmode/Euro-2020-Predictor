About Us

Euro 2020 Predictor is a project started by Chengjie Zhou (Jay Chou). Jay intended to practice his programming skills by writing this project. This project was entirely written in Django, with programming languages like HTML, CSS, Javascript, Python and more. The whole project code for the view function takes about 450 lines. The predictor page, which is emphasized in this project, is a template that predominantly consists of Javascript and HTML. All of the templates and codes can be viewed on github. Click the Github button in the footer to access the link.



The Euro 2020 Predictor Web App is a platform that allow users to predict the result of the EURO 2020 Tournament. Users are able to drag countries to predict the order of each group in the group stage section, then choose the winner of each elimination round and pick the final winner of the tournament in the end. All of these datas can be stored inside the database, which enables user to view their predictions later. Users are also eligable to download a pdf file of their predictions by simply pressing the download button after their prediction or in their prediction pages.
